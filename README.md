# watermeter 水表

这个分支是研究生的分支，功能对比主分支(本科生分支)会有大幅度删减。

(watermeter 是核心代号，项目最终成品将会是 Traintime PDA)

先说个事，eMule 是 eDockey 2000 的开源重写版。那么说，我这个程序就是[这个](https://myxdu.moefactory.com/)的开源重写版咯。  
(感觉我这名字挺能暗示的)

还在开发中，也没有终点。

# 我想要实现什么

1. 研究生服务，包括查成绩，查课表。
2. 支付功能。
